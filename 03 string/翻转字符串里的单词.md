# 翻转字符串里的单词
> Author ID.9276

## 01 题目

给定一个字符串，逐个翻转字符串中的每个单词。

 

**示例 1：**

```
输入: "the sky is blue"
输出: "blue is sky the"
```

**示例 2：**

```
输入: "  hello world!  "
输出: "world! hello"
解释: 输入字符串可以在前面或者后面包含多余的空格，但是反转后的字符不能包括。
```

**示例 3：**

```
输入: "a good   example"
输出: "example good a"
解释: 如果两个单词间有多余的空格，将反转后单词间的空格减少到只含一个。
```

 

**说明：**

- 无空格字符构成一个单词。
- 输入字符串可以在前面或者后面包含多余的空格，但是反转后的字符不能包括。
- 如果两个单词间有多余的空格，将反转后单词间的空格减少到只含一个。

 

**进阶：**

请选用 C 语言的用户尝试使用 *O*(1) 额外空间复杂度的原地解法。

## 02 分析

注意，去空格操作应该在翻转之后进行，应为这样会动态改变 string 的长度！

class Solution {
public:
    string reverseWords(string s) {
        
        if(s.empty()) return s;
        
        // 1. remove space mark.
        removeSpace(s);
        
        // 2. inverse the string 
        inverse(s, 0, s.size()-1);
        
        // 3. reverse the word
        int i = 0;
        int j = i + 1;
        for( ; j < s.size(); j++){
            if(s[i] != ' ' && s[j] == ' ') {
                inverse(s, i, j-1);
                i = j;
            } else if(s[i] == ' ' && s[j] != ' ') i = j;   
        }
        
        // deal the last word.
        if(j == s.size()) inverse(s, i, j-1);
        
        return s;
    }
    
    void removeSpace(string& s) {
         // add front and end mark to remove front and end spaces.
        s = "a " + s + " a";
        
        // 1. remove the space, first and list char is 'a'
        int i = 0;
        bool flag = 0; // space flag
        
        for(int j = 0; j < s.size(); ) {
            if(s[j] != ' ') {
                flag = 1;
                s[i++] = s[j++];
            } else {
                if(flag == 1) {
                    s[i++] = ' ';
                    flag = 0;
                }
                j++;
            }
        }
        
        // clean the string.
        s.erase(i, s.size() - i);
        
        // debug the "    " case.
        if(s == "a a") s = "";
        else {
            // remove front and end mark.
            s.erase(0, 2);
            s.erase(s.size()-2, 2);
        }
    }
    
    void inverse(string& s, int i,int j) {
        
        char tmp;
        
        while(i < j) {
            tmp = s[i];
            s[i] = s[j];
            s[j] = tmp;
            i++;
            j--;
        }
    }
};

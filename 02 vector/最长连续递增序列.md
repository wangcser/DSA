# 最长连续递增序列
**问题关键词：**

- 顺序扫描

## 01 题目

给定一个未经排序的整数数组，找到最长且**连续**的的递增序列。

**示例 1:**

```
输入: [1,3,5,4,7]
输出: 3
解释: 最长连续递增序列是 [1,3,5], 长度为3。
尽管 [1,3,5,7] 也是升序的子序列, 但它不是连续的，因为5和7在原数组里被4隔开。 
```

**示例 2:**

```
输入: [2,2,2,2,2]
输出: 1
解释: 最长连续递增序列是 [2], 长度为1。
```

**注意：**数组长度不会超过10000。

## 04 核心代码

```c++
class Solution {
public:
    int findLengthOfLCIS(vector<int>& nums) {
        
        if(nums.size() < 1) return 0;
        
        int subLen = 1;
        int ans = 1;
        
        for(int i = 1; i < nums.size(); i++) {
            if(nums[i] <= nums[i-1])
                subLen = 1;
            else
                subLen++;
            
            ans = max(ans, subLen);
        }
        
        return ans;
    }
};
```

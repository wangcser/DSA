## 01 题目

编写一个算法来判断一个数是不是“快乐数”。

一个“快乐数”定义为：对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和，然后重复这个过程直到这个数变为 1，也可能是无限循环但始终变不到 1。如果可以变为 1，那么这个数就是快乐数。

**示例:** 

```
输入: 19
输出: true
解释: 
12 + 92 = 82
82 + 22 = 68
62 + 82 = 100
12 + 02 + 02 = 1
```

### 3.1 C++

```c++
//version 1.0 0.0%

class Solution {
public:
    bool isHappy(int n) {
        unordered_set<int> happyNum;
        int next = 0;
        
        while(n != 1) {
            happyNum.insert(n);
            next = 0;
            while(n) {
                int p = n % 10;
                next += p * p;
                n = n / 10;
            }
            
            if(happyNum.count(next) > 0) return false;
            // cout<<next<<endl;
            n = next;
        }
        
        return true;
    }
};
```

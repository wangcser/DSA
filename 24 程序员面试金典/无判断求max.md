## 03 题解

### 基础：inplace 方案-拿出一些元素记录状态

- 第二个表达式中将 b 展开更方便分析
- 注意含有逻辑表达式的括号问题

```c++
class Max {
public:
    int getMax(int a, int b) {
        // write code here
        // b > 0 -> a > b, b < 0 -> a < b
        b = a - b;
        // if a > b, b>>31 = 0. a = a - (a - b) & 0 = a
        // if a < b, b >> 31 = 1, a = a - (a - b) & 1 = b
        a = a - (b & (b >> 31));

        return a;
    }
};

```

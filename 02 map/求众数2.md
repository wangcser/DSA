## 01 题目

给定一个大小为 *n* 的数组，找出其中所有出现超过 `⌊ n/3 ⌋` 次的元素。

**说明:** 要求算法的时间复杂度为 O(n)，空间复杂度为 O(1)。

**示例 1:**

```
输入: [3,2,3]
输出: [3]
```

**示例 2:**

```
输入: [1,1,1,3,3,2,2,2]
输出: [1,2]
```

## 02 输入分析

- 对于这个题目来说，输出最少为 0 个， 最多为 2 个
- 注意对输出的去重

  - 首先用 hash-map 统计
  - 然后遍历 map 取出目标值即可
  - 这里统计和判断要分开进行

## 04 核心代码

```c++
class Solution {
public:
    vector<int> majorityElement(vector<int>& nums) {
        
        map<int, int> m;
        int i = nums.size();
        vector<int> ans;
        
        for(int i = 0; i < nums.size(); i++) {
            m[nums[i]] += 1;
        }
        
        for(auto it : m) {
            if(it.second > nums.size() / 3)
                ans.push_back(it.first);
        }
        
        return ans;
    }
};
```

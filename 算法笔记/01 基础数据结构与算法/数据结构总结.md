# 常用数据结构总结

## 常用数据结构操作效率表

| 数据结构   | 查找    | 插入    | 删除    | 遍历 |
| ---------- | ------- | ------- | ------- | ---- |
| 数组       | O(N)    | O(N)    | O(N)    | O(N) |
| 有序数组   | O(logN) | O(N)    | O(N)    | O(N) |
| 单向链表   | O(N)    | O(1)    | O(N)    | O(N) |
| 有序链表   | O(N)    | O(N)    | O(N)    | O(N) |
| 二叉树     | O(logN) | O(logN) | O(logN) | O(N) |
| BST        | O(logN) | O(logN) | O(logN) | O(N) |
| BBST       | O(logN) | O(logN) | O(logN) | O(N) |
| 图         | O(N)    | O(N)    | O(N)    | O(N) |
| 哈希表     | O(1)    | O(1)    | O(1)    | O(N) |
| 栈         | O(N)    | O(1)    | O(1)    | O(N) |
| 队列       | O(N)    | O(1)    | O(1)    | O(N) |
| 堆         | O(N)    | O(logN) | O(logN) | O(N) |
| 优先级队列 | O(N)    | O(lN)   | O(1)    | O(N) |
|            |         |         |         |      |

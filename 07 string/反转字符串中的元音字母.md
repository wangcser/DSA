# 反转字符串中的元音字母
问题关键词：

- 双指针

## 01 题目

编写一个函数，以字符串作为输入，反转该字符串中的元音字母。

**示例 1:**

```
输入: "hello"
输出: "holle"
```

**示例 2:**

```
输入: "leetcode"
输出: "leotcede"
```

**说明:**
元音字母不包含字母"y"。

## 03 题解

### 基础：双指针

```c++
//version 1.0 0.0%
class Solution {
public:
    string reverseVowels(string s) {
        int i = 0;
        int j = s.size() - 1;
        
        while(i < j) {
            if(isNotVowel(s[i])) i++;
            else if(isNotVowel(s[j])) j--; 
            else {
                swap(s, i, j);
                i++;
                j--;
            }
        }
        
        return s;
    }
    
    bool isNotVowel(char a) {
        if(a == 'a' || a == 'e' || a == 'i' || a == 'o' || a == 'u' ||
           a == 'A' || a == 'E' || a == 'I' || a == 'O' || a == 'U') return false;
        else return true;
    }
    
    
    
    void swap(string&s, int i, int j) {
        int tmp;
        tmp = s[j];
        s[j] = s[i];
        s[i] = tmp;
    }
};
```
